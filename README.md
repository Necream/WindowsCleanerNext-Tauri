# Windows Cleaner Next - Tauri

基于 Tauri 2 的 Windows 系统清理工具，从 Electron 迁移而来。

## 功能

- 清理 Windows 更新缓存
- 清理系统临时文件
- 清理系统日志
- 清理系统还原点
- 清理预取缓存
- 内存缓存清理
- 用户临时文件清理
- 自定义脚本清理
- 浏览器缓存清理
- 应用缓存清理

## 技术栈

- Tauri 2 (Rust)
- Vite
- C++ 原生清理工具

## 运行

```bash
npm install
npm run tauri dev
```

## 构建

```bash
npm run tauri build
```
