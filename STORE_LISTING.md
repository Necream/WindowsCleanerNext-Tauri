## App name

Windows Cleaner Next - Tauri

## Short description

A local-first Windows cleaning tool with a sidebar UI, bundled tools, and optional utility links.

## Long description

Windows Cleaner Next - Tauri is a Windows maintenance and cleanup tool built with Tauri. It focuses on a cleaner sidebar-based workflow, local execution, and practical tools for everyday system cleanup.

Highlights:

- Sidebar-based main interface
- System cleanup categories for cleaner task selection
- Built-in tool launchers and utility links
- Local settings and local output logs
- Theme controls and a compact settings page

This app is designed to run locally on your device. It does not require a cloud account for normal use.
